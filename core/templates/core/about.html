About NovaMart

NovaMart is a personal project I built to explore full-stack development and create a modern e-commerce experience with a multi-vendor system.

🔐 Authentication & Roles

Secure login and signup with role-based access for customers and vendors.

🛒 Shopping Cart

Add, remove, and update products in a real-time cart with count updates.

🏬 Vendor Stores

Vendors can manage their stores, upload products, and track orders easily.

🎨 Modern UI

Responsive design with Tailwind CSS for a sleek shopping experience.

⚙️ Tech Stack

Backend: Django (Python), Django ORM
Frontend: Django Templates, Tailwind CSS
Database: SQLite / MySQL
Other: Authentication, Media uploads, Role management

📚 Learning Outcomes

  • Designed scalable Django models for multi-vendor e-commerce
  • Implemented role-based access control
  • Worked with Tailwind for responsive UI
  • Handled media uploads and product images
  • Built a real-world styled project from scratch
